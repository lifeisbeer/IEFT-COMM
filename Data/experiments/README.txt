-data: contains all the data produced in subfolders depending on the experiment type
-overall results: contains summative results of all experiments in separate text files
  for each experiment type
-BSE_experiments.py: contains the code used to run the experiments (same version as the
  one that produce the data can't be guaranteed but should be pretty similar at least)
-stats.py, stats2.py: contain code used to run hypothesis testing (results found in overall)
